== Description ==

(Description of plugin)


= Usage =

Place these shortcodes in the page, post or text widget where you'd like to display the search form and results.

`
[sanctions-search]
[sanctions-search-results]
`


= Templates =

Place this shortcode in any template parts of your theme.

`
<?php echo do_shortcode('[sanctions-search]'); ?>
<?php echo do_shortcode('[sanctions-search-results]'); ?>
`


== Installation ==
= Installation =
1. Upload "sanctions-search" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
